This is your first start of the digitizer. You need to go through 5 steps, to customize the setting. Just follow the instructions:

  1. Define a reference image on which the digits, counters and reference points are marked
  2. Define two references, which are used to align the image and identify fixed positions on the image
  3. Mark and define the digits to recognize
  4. Mark and define the analog counters to be recognized
  5. Adopt general settings

Just use the buttons "Next" and "Previous" to nagivate through the process.
